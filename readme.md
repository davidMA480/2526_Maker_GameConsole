# Game Console Retro

## Avant propos 

Ce projet a été réalisé dans le cadre de **l'option Maker à l'ENSEA**. Il consiste à réaliser une console rétro sur la base d'une **Raspberry Pi**. J'ai toujours voulu avoir une console de jeu, mais faute d'avoir des parents asiatiques, je n'en ai jamais eu (je les aime quand même). Je profite donc de cette occasion pour développer la mienne et rattraper le temps perdu.

A propos de la console, je veux qu'elle soit portable dans le style de ce qui fait sur le marché (je pense notamment à la console R36S, photo ci-dessous).
